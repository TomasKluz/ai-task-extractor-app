# AI Task Extractor

AI-powered application that transforms meeting notes, emails and free-form text into actionable tasks and events.

## Features

* Extract tasks from text using OpenAI
* Detect events and deadlines
* Store analysis history in PostgreSQL
* Browse previous analyses
* Modern React frontend
* Type-safe backend using TypeScript and Prisma

## Tech Stack

### Backend

* Node.js
* Express
* TypeScript
* PostgreSQL
* Prisma
* OpenAI API

### Frontend

* React
* Vite
* Tailwind CSS
* shadcn/ui
